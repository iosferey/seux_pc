# UX Cultural Evaluator

Sistema para evaluar sitios web utilizando:

- Heurísticas de Nielsen
- Dimensiones culturales (Hofstede y Trompenaars)
- Análisis automatizado con Selenium + GPT

## Uso

python main.py https://example.com

## Resultado

El sistema calcula:

- ICH (Índice de Calidad Heurística)
- IAC Origen (alineación cultural del sitio)
- IAC México (simulación cultural)
- Brecha de transferencia cultural
